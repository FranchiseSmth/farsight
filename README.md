## 📋 About farsight

farsight is a tool designed to help founders and curious minds get detailed research reports quickly. You simply ask a question, and farsight uses artificial intelligence to gather and analyze information for you. It combines different skills powered by Claude, an advanced AI, to create deep reports automatically. This app runs on Windows and is open-source, meaning the code is open for anyone to check or improve.

The app focuses on research in a way that anyone can use, even without technical knowledge. It works with modern technologies like web scraping to collect data and Next.js for a smooth user interface. farsight is written in TypeScript and built to be flexible with pluggable modules.

## ⚙️ Using farsight

Once installed and running, using farsight is simple:

1. **Enter your question:** In the main input box, type a question you want researched. For example, “What are the latest trends in AI for startups?”

2. **Start research:** Click the button labeled “Start” or “Ask” to send your question to farsight.

3. **Wait for the report:** The system will collect information, analyze, and generate a detailed research report. This may take a few moments, depending on your internet speed and the complexity of your question.

4. **View results:** The report will appear on your screen. You can scroll through or save it as a file for later.

5. **Ask more questions:** You can enter new questions anytime, and farsight will create fresh reports for each.

## 🛠 Troubleshooting

If you run into issues, try these steps:

- **Installer won’t run:** Check if your antivirus or Windows Defender is blocking the file. Allow permission if needed.

- **App crashes on start:** Make sure your Windows is updated and meets system requirements.

- **Report generation is slow:** Confirm your internet connection is stable.

- **Report is incomplete:** Try rephrasing your question with clearer terms.

- **General bugs or errors:** Visit the GitHub Issues page on the farsight repository to see current problems or report your own.
